Catch Records from Norway

2015-2019

The following map displays catch records from Norway. The source data (CSV) has been processed, transformed and published as Linked Data, enabling their integration with other datasets in the LOD datasets. The data is already linked with some of these datasets, including FAO species, FAO fishing areas, and the ISO-3166 standard codes. The map shows the first 10000 records. For each record, its identifier and the name of the catched species is displayed, both can be clicked to visualize the full semantic description.
